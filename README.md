# vcs
a version control system in python
## the file structure
```
my_vcs/
  ├── .vcs/
  │   ├── versions/
  │   │   ├── file1_1.0.txt
  │   │   ├── file1_1.1.txt
  │   └── metadata.json
  ├── repository_manager.py
  └── file1.txt
```
## the basic ideas
* initialise the repository
* commit files and track change
* show differences between versions
* display simple metrics
